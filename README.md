💳 Banking System (Java Console Application)
📌 Project Overview
A simple Java Banking System console application that simulates basic banking operations. The system allows users to create bank accounts, deposit money, withdraw money, and view account details.
It is designed to demonstrate core Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and method interaction.

⚙️ Features
Creates a new bank account
View account detail
Deposit money into an account
Withdraw money from an account 

How to Run the Project
Option 1: Using an IDE (Recommended)
Open the project in NetBeans / IntelliJ / Eclipse
Run Main.java
Follow the console menu prompts
Option 2: Using Terminal (if Java is installed)
javac *.java
java Main
